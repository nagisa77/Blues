## A调_90BPM_shuffle.band

- 调性 / 速度：A 调，90 BPM
- 工程：[A 调 `90 BPM` shuffle](./A调_90BPM_shuffle.band)
- 节奏：1 个 chorus 的 shuffle 伴奏，可循环使用
- 即兴 1：自由音符，3 chorus
- 即兴 2：只用 A、C、D，3 chorus
- 对应导出：[节奏基线](../录音/2026-06-18_W01_D1_baseline-rhythm_A_90bpm.mp3)、[自由即兴](../录音/2026-06-18_W01_D1_baseline-solo_A_90bpm.mp3)、[三音限制即兴](../录音/2026-06-18_W01_D1_three-note-solo_ACD_A_90bpm.mp3)

工程内 WAV 由 Git LFS 追踪。更新工程时应复制完整 `.band` 包，并保留导出录音作为不可覆盖的训练证据。
