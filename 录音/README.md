# 录音目录

建议保存未经剪辑的练习录音。它们不是作品，而是判断进步的证据。

## 命名规则

```text
YYYY-MM-DD_W周数_内容_调性_速度.ext
```

示例：

```text
2026-06-15_W01_baseline-solo_A_90bpm.m4a
2026-06-21_W01_week1-performance_A_90bpm.mp4
```

每周至少保留四条；第 1、3、6、9、12 周的周考录音不要覆盖。录完后把路径和“下一遍只改什么”填入[进度看板](../05_进度看板.md#录音索引)。

## 可编辑工程

- [Blues1.band](../工程/Blues1.band)：GarageBand 工程，A 调 90 BPM；包含 1 个 chorus 的 shuffle 伴奏、自由即兴 3 chorus，以及只用 A、C、D 的限制即兴 3 chorus。

工程内的 WAV 素材使用 Git LFS 追踪，导出的练习 MP3 继续保存在本目录。
