# Blues Guitar 大师课训练中心

面向有经验吉他手的 8 周 Blues 强化：以经典曲目为主线，把乐理、手法、名句和完整表演变成同一件事。

默认每周 3 次 `45–60 分钟`主训练，加 1 次不剪辑录音。微练习只在录音暴露出具体问题时介入。

## 现在从这里开始

- D1 四类基线已完成；D2 不再补做重复的基础阶梯，连续表现留给周录音验证。
- D3 已完成：原句跟弹、A 调移调以及“移调原句进入个人即兴”均已归档；结构来自用户自评，录音尚未回听评价，不追加连续三段补考。
- 课程已升级为“真实曲目 + 名句扒谱 + 乐理顿悟 + 手法入句 + 不剪辑表演”；同类 `3 chorus`限制任务每周最多一次。
- 下一步先录第 1 周 A 调完整表演；随后进入[第二周详细计划](./计划/第二周详细计划.md)，主攻 Eric Clapton《Before You Accuse Me》`Unplugged` 版本。

本地参考：[B.B. King - How Blue Can You Get?（Live At The Regal）](./曲目/参考素材/BB_King-How_Blue_Can_You_Get-Live_At_The_Regal.mp3)

> 当前唯一重点：五轮 12 小节始终不丢位置，并让节奏角色自然交给 solo，最后有明确 ending。

## 下一次 30–40 分钟任务

1. A 调 `65 BPM`：`count-in + 节奏 2 chorus + solo 3 chorus + ending`。
2. Solo 依次为主题、变化、发展；可调用已经完成的 B.B. King L01，但不再单独加工它。
3. 热身最多 10 分钟，正式录音只录一遍；失误继续，不局部补录。

过关只看：五个 chorus 中途不停、不丢位置、ending 明确。演奏质量等可靠回听后再判断。

## 8 周完成标准

- 在 A、E、G 三个常用调中，不看谱完成节奏、填句和 solo 的切换。
- 两个完整扒谱 chorus，至少 8 个能在新语境中使用的个人化句子。
- 三首不同 feel 的曲目可不中断演奏，并有明确 intro、solo 入口和 ending。
- 完成一条 `20–25 分钟`不剪辑 set；失误后能继续，groove 不散。
- 推弦、揉弦和静音达到“录音中不抢走注意力”的实用标准。

乐理解释声音，手法改变语气，名句提供语法，录音才是证据。
