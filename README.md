# Blues Guitar 大师课训练中心

面向有经验吉他手的 8 周 Blues 强化：以经典曲目为主线，把乐理、手法、名句和完整表演变成同一件事。

默认每周 3 次 `45–60 分钟`主训练，加 1 次不剪辑录音。微练习只在录音暴露出具体问题时介入。

## 下一次 25–30 分钟任务

1. 听 [Eric Clapton - Before You Accuse Me `Unplugged`](./曲目/参考素材/Eric_Clapton-Before_You_Accuse_Me-Unplugged_yt_416wa6CHs04.mp3)，只选一个 `2–4` 小节 rhythm fill 或 turnaround。
2. 在琴上复制原句，写清它在哪几个小节、从哪一拍进入、尾音落在 E7 / A7 / B7 的哪个 guide tone。
3. 只做一个变化：保留节奏轮廓，改最后 `2–3` 个音或尾音长度。
4. 放回 [E 调 `90 BPM` 3 chorus（三轮 12 小节）背景与 solo（主音独奏）工程](./工程/E调_90BPM_3chorus背景与solo.band)：`1 chorus` rhythm（节奏/伴奏）+ 原句 + 变化 + ending（收尾）。

过关只看：能听出它来自 Clapton 原曲；你能说出一个和声理由；放回曲目时 groove（律动）不断。

## 8 周完成标准

- 在 A、E、G 三个常用调中，不看谱完成 rhythm（节奏/伴奏）、fill（填句）和 solo（主音独奏）的切换。
- 两个完整扒谱 chorus（一轮 12 小节），至少 8 个能在新语境中使用的个人化句子。
- 三首不同 feel（律动感觉）的曲目可不中断演奏，并有明确 intro（前奏）、solo（主音独奏）入口和 ending（收尾）。
- 完成一条 `20–25 分钟`不剪辑 set（曲目组）；失误后能继续，groove（律动）不散。
- 推弦、揉弦和静音达到“录音中不抢走注意力”的实用标准。

乐理解释声音，手法改变语气，名句提供语法，录音才是证据。
